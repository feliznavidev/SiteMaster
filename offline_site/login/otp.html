Multi-Factor Authentication

Multi-Factor Authentication (MFA) enhances security by requiring a physical device and your Canvas login password. The device can be a code generator or a mobile phone that receives text messages. Using an authenticator app is strongly recommended for enhanced security.

Configure MFA by scanning the QR code with a Time-based One-Time Password (TOTP) app on iPhone or Android (e.g., Google Authenticator, Microsoft Authenticator, Authy). After setup, print backup codes from your profile page if your primary MFA device is unavailable.
